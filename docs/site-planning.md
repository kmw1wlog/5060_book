# 5060 고전 서적 판매 웹사이트 기획안

## 1. 프로젝트 목적 이해

이 프로젝트의 핵심은 단순한 "저가 책 판매"가 아니다.

- 5060 세대가 부담 없이 고전을 다시 읽게 만드는 입문형 실물 서점
- 경제학, 철학, 역사 분야의 거장 텍스트를 오늘의 한국어로 다시 연결하는 브랜드
- 두꺼운 책도 평균 5천원대라는 가격 장벽 해소
- "AI 번역"이 아니라 "신뢰 가능한 번역 공정"을 공개해 구매 불안을 없애는 구조
- 책 1권 판매보다 세트 구매, 재구매, 독서모임 참여까지 이어지는 관계형 서비스

즉, 이 사이트는 책을 예쁘게 보여주는 것이 목적이 아니라 아래 3가지를 동시에 달성해야 한다.

1. 믿을 수 있다
2. 읽을 수 있겠다
3. 지금 사도 부담 없다

## 2. 핵심 타깃

### 1차 타깃

- 50대 후반~60대 초반
- 은퇴 전후, 혹은 자영업/전문직/직장 경력을 가진 독자
- 젊을 때 못 읽었던 고전을 지금 다시 읽고 싶지만, 원전 번역의 난이도와 가격이 부담인 층

### 2차 타깃

- 40대 후반~50대 초반
- 경제, 정치, 인문 교양에 관심은 있지만 입문 동선이 필요한 독자
- 부모 세대 선물용으로 구매하는 자녀층

### 타깃의 실제 구매 조건

- 글자가 작으면 이탈한다
- 번역이 난해하면 재구매하지 않는다
- 책이 많으면 무엇부터 읽어야 할지 몰라 멈춘다
- 번역 품질과 판본 출처가 불명확하면 의심한다
- 비싸면 "언젠가"로 미룬다

## 3. 브랜드 포지셔닝

### 브랜드 한 줄 정의

거장의 원전을, 오늘의 한국어와 합리적인 가격으로 다시 읽게 만드는 서점

### 구매 설득 문장

- 거장의 원전을 오늘 읽을 수 있게 바꿨습니다.
- 어려운 고전을 큰 글씨, 검증된 번역, 낮은 가격으로 제공합니다.
- 한 권부터 시작해도 되고, 질문별 세트로 바로 시작해도 됩니다.

### 브랜드 성격

- 지적이지만 차갑지 않음
- 고전적이지만 올드하지 않음
- 품위 있지만 비싸 보이지 않음
- 설명적이지만 과장되지 않음

## 4. 디자인 방향

첫 페이지 기준으로 전체 사이트의 시각 원칙을 통일한다.

### 비주얼 키워드

- 서재
- 고전
- 신뢰
- 여백
- 온기
- 품격 있는 실용성

### 스타일 원칙

- 배경은 완전한 흰색보다 따뜻한 아이보리 계열 사용
- 메인 컬러는 짙은 녹색과 브론즈/골드 포인트 조합
- 카드와 박스는 얇은 보더, 라운드, 약한 그림자로 정돈
- 사진은 차갑고 상업적인 스톡 느낌보다 서재와 자연광 느낌 유지
- 타이포는 제목에 명조 계열 감성, 본문은 가독성 높은 산세리프 조합
- 과한 모션보다 부드러운 페이드, 슬라이드, 호버 강조 정도만 사용

### 톤앤매너

- "최저가", "파격", "한정" 중심의 할인몰 어조는 피한다
- "지금 시작해도 늦지 않습니다", "오늘의 한국어로 다시 읽습니다" 같은 안정감 있는 카피 사용
- 5060 세대가 "무시당하지 않는다"는 느낌이 중요하다

## 5. 첫 페이지 기준 정보 구조

사용자가 처음 들어왔을 때 가져가야 하는 흐름은 아래 순서다.

1. 브랜드와 카테고리를 이해한다
2. 왜 이 책이 필요한지 감성적으로 납득한다
3. 가격과 접근성이 낮다는 점을 확인한다
4. 나에게 맞는 시작점을 찾는다
5. 대표 세트 혹은 단권으로 진입한다
6. 번역 품질에 대한 의심을 해소한다
7. 혼자 읽지 않아도 된다는 관계 가치를 본다
8. 무료 독서지도 PDF로 리드를 남긴다

## 6. 화면별 기획

아래 순서는 첨부된 화면 1~8의 핵심 구성과 순서를 유지한 결과다.

### 화면 1. 메인 랜딩

역할: 브랜드 첫인상, 대표 가치 제안, 첫 클릭 유도

구성:

- 상단 헤더
- 메인 히어로 카피
- 대표 도서 세트 비주얼
- CTA 2개
- 핵심 가치 바
- 관심사 기반 3개 진입 카드

의도:

- "거장의 원전을 오늘의 한국어로"를 가장 먼저 각인
- 평균 권당 5천원대라는 가격 메시지를 조기 노출
- 대표 컬렉션 보기와 무료 독서지도 받기 두 갈래로 진입 유도

권장 카피:

- 메인 타이틀: 거장의 원전을 오늘의 한국어로
- 서브카피: 5060을 위한 경제·철학 고전 시리즈, 부담 없는 가격으로 내 서재에
- CTA 1: 대표 컬렉션 보기
- CTA 2: 무료 독서지도 받기

### 화면 2. 질문별 시작점

역할: "무엇부터 읽지?" 문제 해결

구성:

- 질문형 헤드라인
- 관심사 카드 4개
- 시작 추천 카드 1개
- 대표 세트 추천 카드 2개

카테고리 추천:

- 돈과 자산
- 삶과 태도
- 사회와 역사
- 처음부터 쉽게

의도:

- 사용자가 책 제목이 아니라 자기 고민으로 탐색하게 만든다

### 화면 3. 대표 컬렉션

역할: 실제 구매 전환 구간

구성:

- 대표 세트 2개
- 세트별 포함 도서, 추천 대상, 가격
- 세트 CTA
- 단권 추천 3개
- 가치 바 재노출

세트 전략:

- 돈을 다시 생각하는 6권 세트
- 삶을 다시 생각하는 6권 세트

가격 전략:

- 세트는 29,900원 전후
- 단권은 4,900원~6,900원대 중심
- "세트 무료배송"을 강하게 표시

### 화면 4. 상품 상세

역할: 한 권 구매 직전의 의심 제거

구성:

- 대형 표지 이미지와 썸네일
- 제목, 저자, 시리즈, 가격
- 핵심 장점 3개
- 장바구니/바로구매
- 왜 지금 읽어야 하는가
- 원문과 새 번역 미리보기
- 목차 미리보기
- 함께 읽으면 좋은 책

핵심 메시지:

- 큰 글씨
- 원문 대조 번역
- 교정·교열 완료

### 화면 5. 번역 원칙

역할: 가장 큰 불안 요소인 번역 신뢰성 해결

구성:

- 번역 프로세스 7단계
- 번역 원칙 박스
- 독자 약속 박스
- 판본 정보 예시 테이블

꼭 들어가야 할 메시지:

- 판본 출처 공개
- 용어 일관성 유지
- 사람의 교정·교열
- 오류 신고 및 수정 이력 반영

주의:

- "AI가 번역했습니다"보다 "AI 초벌 + 편집 검수 + 분야 감수" 구조를 전면화
- 저작권 만료 여부는 도서별 검수 절차로 별도 관리한다고 표현

### 화면 6. 독서모임

역할: 혼자 읽기 어렵다는 장벽 해소

구성:

- 독서모임 메인 비주얼
- 밴드, 유튜브, 카카오 채널 진입 카드
- 독자 후기 3개
- 다음 출간 투표 카드
- 커뮤니티 가치 바

의도:

- 책 판매를 모임 가입으로 연결
- 구매 후 이탈이 아니라 관계 유지 구조 형성

### 화면 7. 주문하기

역할: 결제 이탈 최소화

구성:

- 장바구니 목록
- 배송 정보 입력
- 주문 요약
- 결제 수단 선택
- 상담 박스

원칙:

- 폼 입력 최소화
- 폰트 크기 크게
- 결제 영역 우측 고정
- "세트 무료배송" 반복 노출

### 화면 8. 리드 수집

역할: 바로 구매하지 않는 방문자 회수

구성:

- 무료 혜택 3개
- 이름/휴대전화/관심 분야 입력
- 개인정보 동의
- 무료 독서지도 신청 CTA
- 창립 독자 혜택 소개

제안 리드 자산:

- 거장 20인 독서지도 PDF
- 신간 예약판매 우선 안내
- 다음 출간 투표 참여권

## 7. 첫 페이지 디자인 시스템 제안

### 색상

- Primary: 깊은 서재 녹색
- Secondary: 브론즈 골드
- Background: 웜 아이보리
- Surface: 크림 화이트
- Text: 먹색에 가까운 딥 그린/다크 브라운
- Accent: 차분한 세이지

### 타이포

- 제목: 한국어 명조 계열 또는 고전미 있는 디스플레이 계열
- 본문: 가독성 높은 산세리프
- 숫자/가격: 굵고 크게, 즉시 비교 가능하게

### 컴포넌트 원칙

- 버튼은 채움형 1개, 아웃라인형 1개를 기본 조합으로 사용
- 카드 하단 화살표 버튼은 사이트 전반 공통 스타일 유지
- 배지, 무료배송, 신뢰 요소는 동일한 톤으로 반복 사용
- 아이콘은 라인 기반으로 통일하고 금색/녹색 조합 유지

## 10. 실제 구현 시 주의사항

- 5060 대상이므로 모바일 대응은 하되, 데스크톱 경험을 우선 설계
- 텍스트는 작게 멋내지 말고 명확하게 크게 보여줄 것
- 카드 수를 과하게 늘리지 말고 선택 피로를 낮출 것
- 가격, 무료배송, 번역 신뢰를 모든 주요 화면에서 반복 노출할 것
- 삽입 이미지는 현재 준비된 `src` 자산을 사용하되 색감 보정 없이도 통일감이 나도록 배경색과 그림자 톤을 맞출 것
- 법적 표현은 "공개 가능한 판본 정보 공개", "검수 절차 운영" 수준으로 쓰고, 개별 도서 저작권 검토 프로세스는 별도 운영 문서에서 관리할 것
